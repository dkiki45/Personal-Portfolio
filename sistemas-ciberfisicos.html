Sistemas Ciberfísicos

Nesta seção de Sistemas Ciberfísicos, onde o mundo físico se encontra com o digital.

Tópicos Fundamentais em Sistemas Ciberfísicos

Cálculo de Binários e Fundamentos Digitais

Exploramos a base da computação digital através do sistema binário, conversões entre diferentes bases numéricas (decimal, binário, hexadecimal) e operações lógicas. Compreender esses fundamentos é essencial para trabalhar com microcontroladores e processamento de dados em nível de hardware.

Evolução do Computador

Estudamos a história e a evolução dos computadores, desde as primeiras máquinas de cálculo até os sistemas modernos. Isso nos permitiu entender como a arquitetura e as capacidades dos dispositivos foram aprimoradas, moldando o cenário tecnológico atual.

Evolução das Redes de Computadores

Analisamos o desenvolvimento das redes de computadores, desde suas origens até a internet global. Compreendemos os diferentes tipos de redes, protocolos de comunicação e como os dispositivos se conectam e interagem, um pilar para sistemas distribuídos e IoT.

Projetos Práticos com Arduino (Wokwi)

1ª Atividade: Controle Sequencial de LEDs e Semáforo

Esta atividade focou na programação de LEDs para piscar em padrões específicos e na simulação de um sistema de semáforo, demonstrando controle básico de atuadores e temporização.

  • LED Único (Ligado 1s / Apagado 3s): Controle básico de um LED, definindo estados e temporizações.
  • Dois LEDs com Tempos Independentes: Expansão para dois LEDs, cada um com seu próprio padrão de tempo.
  • Semáforo de 3 LEDs: Simulação de um semáforo de trânsito com sequências lógicas de acendimento.
Ver no Wokwi

2ª Atividade: Leitura de Dados de Temperatura e Umidade (DHT22)

Esta atividade focou na integração e leitura de dados de um sensor ambiental. Utilizamos o sensor DHT22 para adquirir dados de temperatura e umidade, exibindo-os no monitor serial do Wokwi.

Ver no Wokwi

3ª Atividade: Controle de Servo Motor com Dados de Sensor

Esta atividade combinou o controle de um servo motor com a leitura de dados de temperatura e umidade, demonstrando uma aplicação prática de sensores e atuadores. O servo se movimenta com base nos dados lidos do sensor DHT22.

Projeto Final: Soluções para o Mundo

Neste projeto, desenvolvemos com o uso do Arduino uma solução inovadora: um sistema capaz de identificar medições de terremotos e acionar alertas preventivos. Ao detectar o risco, o dispositivo envia notificações para celulares e ativa uma sirene, avisando a população da cidade sobre a iminência do terremoto e a necessidade de se preparar adequadamente.
